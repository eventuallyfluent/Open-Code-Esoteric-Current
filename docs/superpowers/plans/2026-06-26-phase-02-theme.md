# Phase 2 — observatory-index Block Theme Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a custom block theme for The Esoteric Current with academic/intelligence-dossier aesthetics.

**Architecture:** WordPress Full Site Editing (FSE) block theme. Pure presentation layer — no business logic, no private table access. All dynamic content rendered via the plugin's six server-rendered blocks. Styling via theme.json with CSS custom properties for the custom palette.

**Tech Stack:** WordPress block theme API, theme.json, HTML block markup, minimal CSS, no JavaScript build step.

## Global Constraints

- No PHP except functions.php (which must be minimal — register block style variations, enqueue theme styles)
- Block theme architecture: theme.json drives all styling; templates are pure HTML block markup
- Theme slug: `observatory-index`
- Text domain: `observatory-index`
- All plugin blocks (`ec/unified-search`, `ec/editorial-feed`, `ec/resource-index`, `ec/source-record`, `ec/issue-contents`, `ec/submission-form`) must be supported in templates
- Visual palette: black (`#000000`), warm white (`#F5F0E8`), muted gold (`#C8A84E`), deep burgundy (`#6B1D2F`), dark navy (`#1A2238`)
- Typography: serif (Playfair Display or Merriweather) for headings/editorial, sans-serif (Inter or system) for nav/metadata/filters
- Must pass WordPress theme review requirements for block themes

```php
<?php
namespace ObservatoryIndex;

add_action('after_setup_theme', function () {
    add_editor_style('assets/editor.css');
    add_theme_support('wp-block-styles');
    add_theme_support('align-wide');
    add_theme_support('responsive-embeds');

    register_block_style('core/separator', [
        'name' => 'ec-thin-rule',
        'label' => __('Thin Rule', 'observatory-index'),

add_action('wp_enqueue_scripts', function () {
    wp_enqueue_style('observatory-index-google-fonts', 'https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600&family=Playfair+Display:wght@400;600;700&display=swap', [], null);
    wp_enqueue_style('observatory-index-theme', get_template_directory_uri() . '/assets/theme.css', ['observatory-index-google-fonts'], wp_get_theme()->get('Version'));
});
```
